# Trumed App

Mobile App

## Requirements

* iOS 12+ (2018)
* Xcode 12+,14+
* Flutter 3.7+
* Dart SDK 2.19+

## Deploy Locally

1. `git clone ssh://git-codecommit.eu-west-1.amazonaws.com/v1/repos/trumed_app`
2. `cd trumed_app`
3. `flutter pub get`
4. Open project with Visual Studio Code (You must have the plugins for flutter and dart installed.)
5. Open lib/main.dart file
6. Press F5 or Run
7. Select Simulator
